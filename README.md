# ⚽ Panini 2026 Tracker 🚀

Una aplicación interactiva diseñada para gestionar y rastrear la colección de láminas del álbum Panini para el **Mundial 2026**.

### ✨ Características
* **Gestión total**: Soporte para las 48 selecciones del torneo.
* **Lógica de grupos**: Clasificación automática por grupos y estadísticas en tiempo real.
* **Interfaz optimizada**: Diseñada para ser usada fácilmente desde el celular.

---
*Proyecto desarrollado con React y GitHub Actions.*.
